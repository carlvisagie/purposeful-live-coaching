## Part 2: Rode NT1 + Focusrite Scarlett Setup

### Why This Combo is Perfect for Coaching
- **Rode NT1:** Studio-quality condenser mic with ultra-low noise
- **Focusrite Scarlett:** Professional audio interface with pristine preamps
- **Result:** Crystal-clear voice quality that builds trust with clients

### Physical Setup

1. **Connect Microphone to Interface**
   - Plug Rode NT1 XLR cable into Focusrite Scarlett Input 1
   - Turn on **48V Phantom Power** (required for condenser mics)
   - Look for red light on Scarlett indicating phantom power is ON

2. **Connect Interface to Computer**
   - Plug USB cable from Scarlett to computer
   - Wait for drivers to install (automatic on Windows/Mac)
   - Scarlett will show up as audio device

3. **Connect Headphones**
   - Plug headphones into Scarlett headphone jack
   - This prevents echo/feedback during calls

### Computer Audio Settings

**Windows:**
1. Right-click speaker icon â†’ Sound settings
2. **Input:** Select "Focusrite Scarlett" or "Scarlett 2i2"
3. **Output:** Select "Focusrite Scarlett" (for headphones)
4. Test microphone - speak and watch input level meter

**macOS:**
1. System Preferences â†’ Sound
2. **Input tab:** Select "Scarlett 2i2 USB"
3. **Output tab:** Select "Scarlett 2i2 USB"
4. Adjust input volume slider

### Focusrite Scarlett Settings

1. **Gain Knob (Input 1):**
   - Start at 12 o'clock position
   - Speak at normal volume
   - Green light = good level
   - Yellow light = getting loud
   - Red light = too loud (turn down gain)
   - **Target:** Green with occasional yellow peaks

2. **Monitor Knob:**
   - Controls headphone volume
   - Adjust to comfortable listening level

3. **Direct Monitor Switch:**
   - **OFF** for video calls (prevents echo)
   - **ON** for recording (hear yourself in real-time)

5. Enable "Automatically adjust microphone volume" (optional)

**Google Meet:**

## Part 3: Pre-Session Testing Checklist

### 30 Minutes Before Client Session

1. **Power On Equipment**
   - [ ] Insta360 Link connected and recognized
   - [ ] Focusrite Scarlett connected (green lights on)
   - [ ] 48V Phantom Power ON for Rode NT1
   - [ ] Headphones plugged into Scarlett

2. **Test Video**
   - [ ] Open video platform (Zoom/Meet/etc.)
   - [ ] Camera shows your face clearly
   - [ ] Lighting is good (face visible, not backlit)
   - [ ] Auto-tracking works (if enabled)

3. **Test Audio**
   - [ ] Speak into Rode NT1 at normal volume
   - [ ] Scarlett shows green/yellow lights (not red)
   - [ ] Record 10-second test - play back
   - [ ] Voice sounds clear, no echo, no distortion
   - [ ] Headphones work (hear yourself clearly)

4. **Browser Permissions**
   - [ ] Camera permission granted
   - [ ] Microphone permission granted
   - [ ] No other apps using camera/mic

5. **Internet Connection**
   - [ ] Speed test: >5 Mbps upload (for HD video)
   - [ ] Close bandwidth-heavy apps
   - [ ] Use wired ethernet if possible (more stable than WiFi)

## Part 4: Optimal Settings for Coaching Sessions

### Recommended Configuration

**Video Quality:**
- Resolution: 1080p (balance of quality and bandwidth)
- Frame rate: 30 fps
- Enable HD in platform settings

**Audio Quality:**
- Sample rate: 48 kHz (Scarlett default)
- Bit depth: 24-bit
- Disable noise suppression (your mic is already studio-quality)
- Disable echo cancellation (headphones prevent echo)

**Microphone Technique:**
- Position mic 6-8 inches from mouth
- Speak directly toward mic (not off-axis)
- Avoid touching desk (vibrations transfer to mic)
- Use pop filter if available (reduces plosives)

## Part 5: Troubleshooting

### Camera Issues

**Camera not detected:**
- Unplug and replug USB cable
- Try different USB port
- Restart Insta360 Link Controller app
- Restart browser

**Poor video quality:**
- Check internet speed (need >5 Mbps upload)
- Lower resolution to 720p
- Close other apps using camera
- Ensure good lighting

**Auto-tracking not working:**
- Open Insta360 Link Controller app
- Enable "Auto Tracking" feature
- Ensure you're in frame when it starts

### Audio Issues

**No sound from microphone:**
- Check 48V phantom power is ON (red light on Scarlett)
- Check XLR cable is fully plugged in
- Increase gain knob on Scarlett
- Select correct input in platform settings

**Echo or feedback:**
- Use headphones (don't use speakers)
- Turn Direct Monitor OFF on Scarlett
- Lower speaker/headphone volume

**Distorted/clipping audio:**
- Turn down gain knob on Scarlett (aim for green lights)
- Move mic further from mouth
- Speak at normal volume (don't shout)

**Very quiet audio:**
- Turn up gain knob on Scarlett
- Move mic closer (6-8 inches optimal)
- Check platform input level settings
- Ensure 48V phantom power is ON

**Buzzing or humming:**
- Use different USB port
- Plug Scarlett into different power outlet
- Move away from electrical interference
- Check XLR cable is not damaged

## Part 6: First Client Session Protocol

### 15 Minutes Before

1. **Run full equipment test** (checklist above)
2. **Close unnecessary apps** (free up bandwidth)
3. **Silence phone** (no interruptions)
4. **Test session link** (make sure it works)
5. **Have backup plan** (phone number if tech fails)

### During Session

1. **Monitor audio levels** (glance at Scarlett lights occasionally)
2. **Stay in frame** (let auto-tracking do its job)
3. **Mute when not speaking** (if needed)
4. **Watch for technical issues** (lag, freezing, audio drops)

### After Session

1. **Ask client about audio/video quality**
2. **Note any issues** for next time
3. **Adjust settings** if needed
