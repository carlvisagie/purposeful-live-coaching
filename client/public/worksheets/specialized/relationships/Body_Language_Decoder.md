# Body Language Decoder

**Module:** Relationships

**Purpose:** To help you to understand and interpret non-verbal cues in your interactions with others.

**Instructions:** This guide provides a list of common body language cues and their possible meanings. Use this guide to become more aware of the non-verbal signals that you and others are sending.

---

### Common Body Language Cues

| Cue | Possible Meaning(s) |
| :--- | :--- |
| **Crossed Arms** | Defensive, closed-off, insecure |
| **Open Arms** | Open, receptive, confident |
| **Leaning In** | Interested, engaged, attracted |
| **Leaning Away** | Disinterested, disengaged, repelled |
| **Making Eye Contact** | Confident, honest, interested |
| **Avoiding Eye Contact** | Shy, insecure, dishonest |
| **Nodding** | Agreement, understanding, encouragement |
| **Shaking Head** | Disagreement, disbelief, disapproval |
| **Smiling** | Happiness, friendliness, amusement |
| **Frowning** | Sadness, anger, disapproval |
| **Raising Eyebrows** | Surprise, disbelief, curiosity |
| **Furrowing Eyebrows** | Confusion, concentration, anger |
| **Touching Face** | Nervousness, anxiety, dishonesty |
| **Fidgeting** | Boredom, nervousness, impatience |
| **Mirroring** | Rapport, connection, empathy |
